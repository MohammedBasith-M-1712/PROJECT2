# Predictive Modeling Using Machine Learning

## Project Overview
This project uses Machine Learning techniques to predict student performance based on study hours, attendance, assignment scores, and previous exam scores.

A Decision Tree Classifier is used to train and test the model.

---

# Dataset Name
Student_Performance_Prediction.csv

---

# Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

---

# Features

- Load and preprocess dataset
- Train Machine Learning model
- Predict student results
- Evaluate model accuracy
- Visualize confusion matrix

---

# Machine Learning Algorithm

- Decision Tree Classifier
